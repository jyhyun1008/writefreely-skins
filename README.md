# Writefreely Skins

| No. | Skin Title | Description |
|---|---|---|
| 0 | Hiyuno | A cute and simple blog theme. |
| 1 | Multiblog | It will be a good theme if you have multiple writefreely blog and want to integrate them! You can use your blog as first subcategory and blog tags as second subcategory! |
| 2 | Daydream | A theme looks similar to the mastodon instance [daydream.ink](https://daydream.ink) 's theme. You can Add your own 200*40 sized banner and banner code! |
